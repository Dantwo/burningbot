Burningbot is a community maintained project. The bot is written in Clojure and uses the irclj to communicate with the server. The Enlive library powers the bots scraper and is used for templating on the web application. Database (MySQL) access is via ClojureQL and clojure.contrib.sql. The web interface is built on Ring and Moustache. jQuery and Modernizr are used to do most of the heavy lifting on the front end.

Initial development by Andrew Brehaut with feedback and bug hunting from Zelbanian, Cathexis, PeregrinePDX, Lira and oliof. You can contribute via the projects GitHub page.

The web design is by Andrew Brehaut and uses the Griffos typeface by Manfred Klein. The paper texture is Exclusive paper from Subtle Patterns.

About Burningbot

Burningbot is an IRC bot for the (unofficial) #burningwheel (and related) IRC channels. It is intended to support activity in the channel and provide a community resource to access archived discussions.
